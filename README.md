# Python Basics Learning Folder

This folder contains basic Python examples and exercises to help you learn Python programming fundamentals.

## Getting Started

1. A Python virtual environment has been set up in `.venv/`
2. Run the examples using: `python filename.py` (the environment is already configured)
   - Or manually: `.venv\Scripts\python.exe filename.py` on Windows

## Contents

- `hello_world.py`: Your first Python program
- `variables.py`: Introduction to variables
- `data_types.py`: Basic data types in Python
- `control_flow.py`: If statements and loops
- `functions.py`: Defining and using functions

## Exercises

Try modifying the code and experimenting with different values to understand how Python works!
